# План: Презентация рекомендаций по улучшению Delta

## Цель
Создать визуальную презентацию (HTML/CSS) с демонстрацией рекомендуемых изменений, чтобы вы могли оценить и выбрать что реализовать.

## Формат
Одностраничная HTML-презентация с:

### Слайд 1: Текущее состояние vs Новое
- Скриншот текущего sidebar + карточек (описание)
- Ключевые проблемы: нет статус/приоритет Select, нет inline editing

### Слайд 2: Карточки с полным функционалом
- Mock-up карточки с:
  - Select статуса (цветной бейдж)
  - Select приоритета (цветной бейдж)
  - Inline editing имени (клик → textarea)
  - Inline editing номера (клик → input)
  - Inline editing Plan/Fact часов (клик → input)
  - Кнопка архива комментариев 📜
  - Progress bar с цветом

### Слайд 3: Беклог в карточном стиле
- Mock-up беклога с карточками вместо таблицы
- Queue badges (1-3 красный/оранжевый/жёлтый)
- Drag handle для reorder

### Слайд 4: Синхронизация темы sidebar
- Текущий sidebar (серый shadcn) vs Sidebar с tracker-темой
- Accent цвет на активном пункте
- Фон sidebar = tracker-bg-card

### Слайд 5: Поиск и удаление
- Поиск вынесен из dropdown в отдельное поле
- Confirmation dialog при удалении задачи

### Слайд 6: Keyboard shortcuts
- Таблица: Ctrl+1-7 → view, Ctrl+F → поиск, Escape → закрыть

### Слайд 7: Итого — приоритеты
- Высокий: #4-7 (карточки полные)
- Средний: #1 (беклог), #15 (тема), #9,13 (поиск+удаление)
- Низкий: #2 (превью), #12 (.shortcuts)

## Файлы
- Создать: `public/recommendations.html` — одностраничная презентация
- Использовать: существующие CSS-переменные tracker-темы для стилизации

## Верификация
- Открыть `public/recommendations.html` в браузере
- Убедиться что mock-upы наглядно показывают изменения
