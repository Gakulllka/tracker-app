## Часть 1: Кнопка «Зайти как гость»

### 1.1 API endpoint: `src/app/api/auth/guest/route.ts` (новый файл)
- Создаёт/находит пользователя `guest` с ролью `admin`
- Если пользователь `guest` не существует — создаёт его через Prisma (username: `guest`, password: `guest123`, displayName: `Гость`, role: admin)
- Если существует — просто выдаёт сессию
- Возвращает `{ success, user, token, workspaceId }` — тот же формат что `/api/auth/login`

### 1.2 Auth Screen: `src/components/auth-screen.tsx`
- Добавить кнопку «Зайти как гость» под основной кнопкой «Войти»
- Кнопка: outline стиль, иконка UserCheck, вызывает `POST /api/auth/guest`
- При успехе — вызывает `onAuth()` с данными гостя
- Визуально: отдельная секция с divider, серый outline кнопка
